About Me

My journey started in my first tech job out of college as a logistics dispatcher for Google Express. I soon realized how powerful data was and my curiosity drove me into teaching myself SQL as my first foray into the data world. Soon after I enrolled in a Data Analytics & Visualization Bootcamp offered by UC Berkeley Extension. I was exposed to web development with micro webframeworks like Flask, Web Scraping Tools utilizing Python, creating databases with MongoDB & MySql, data visualizations with Tableau and so much more.

This propelled me into a position at Uber where I was the main Analytics & Insights Support for Safety Support Program. In this role, I was able to spearhead a data engineering project where I built and launched the first set of safety support data tables that is now used as the main source of truth across the globe by all regional analysts. Seeing how powerful and important ingesting data into a structured and modeled state for users confirmed my interest in pursuing a data engineering career.

I live by these rules:

Stay humble, Be curious, Never stop learning
Focus on your own journey and not others
